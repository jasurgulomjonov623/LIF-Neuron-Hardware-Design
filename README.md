# Neuromorphic LIF Neuron Hardware

Design, simulation, and PCB implementation of an analog Leaky Integrate-and-Fire (LIF) neuron circuit for neuromorphic computing and spiking neural network (SNN) research.

---

## Project Overview

This project focuses on implementing a biologically inspired LIF neuron using analog hardware techniques. The circuit demonstrates membrane potential integration and spike-generation behavior used in neuromorphic systems and SNN architectures.

The circuit uses resistive elements as configurable synaptic weights. During simulation, specific resistance values were selected to analyze neuron behavior and spike-generation characteristics.

For the PCB implementation, resistor positions were designed with pin headers, allowing resistors to be easily replaced and tested with different weight values. This enables flexible experimentation with various resistance configurations and neuron responses.

---

## Features

- Analog LIF neuron implementation
- Configurable resistive weight design
- Replaceable resistor headers for testing
- PCB implementation
- Circuit simulation
- Spike-generation behavior
- Low-power neuromorphic hardware concepts
